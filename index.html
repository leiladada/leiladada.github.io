Les bases pour créer son propre livre numérique

Introduction

Pourquoi cette formation ?

A travers la lecture de cet ouvrage, vous découvrez le monde de l’édition numérique. Cette formation – dont les étapes se trouveront à la fin de chaque chapitre – vous permettra de passer à la pratique : concevoir vous-même un livre numérique.
Nous découvrirons ensemble les bases du format ePub ainsi que les notions fondamentales des langages html & css qui vous permettront de gérer à la fois la structure et la mise en forme de votre livre.

Le matériel nécessaire

Pour cette formation, aucun prérequis particulier n'est nécessaire. En revanche, vous aurez besoin de quelques outils préables. Les voici :

  1. Une connexion internet
  2. Un éditeur de texte. Si vous ne savez pas lequel choisir, je vous conseille celui-ci par exemple.
  3. Le logiciel Sigil
    En téléchargement ici
  4. Un court texte qui vous servira de support de travail (de préférence avec quelques images)
    Conseil : si vous n'en avez, vous pour toujours sélectionner un article qui vous intéresse sur Wikipédia !

Mise en place

La mise en place est assez simple :

  1. Installer le logiciel Sigil sur votre ordinateur
  2. Réserver votre texte dans votre traitement de texte favori

Ce qu'il faut savoir avant de commencer

Quelques informations nécessaires pour comprendre les technologies que nous allons utiliser :

  1. Le langage HTML est un langage de balisage qui permet de structurer la sémantique de nos textes. Ce langage est d’abord utilisé pour le web. Mais il se trouve que les livre numérique au format epub sont également structurés en html. Cela signifie que ce langage nous servira à structurer les différentes parties de notre texte. Nous ne verrons donc de ce langage que ce qui est utile pour fabriquer un livre numérique.
  2. Le langage css consiste en une feuille de style sur laquelle nous allons donner des instructions de mise en page à nos éléments textuels. Par exemple, nous pourrons y déterminer la couleur de nos titres, ou encore les styles de nos polices.
  3. Cette formation étant centrée sur la pratique, je vous conseille néanmoins de suivre les liens hypertextes afin d’approfondir les notions que nous allons parcourir.
